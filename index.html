Tentang

Web ini merupakan sebuah halaman utama yang terinspirasi dari web HoyoLAB yang sering saya kunjungi setiap harinya. Selain untuk membaca postingan Official dari Hoyoverse, saya pun mengunjunginya untuk melihat berbagai postingan yang dibuat oleh berbagai user di kolom forum. Melalui postingan mereka saya dapat menemukan informasi seperti guide alias arahan untuk menyelesaikan berbagai goals yang ingin saya capai di dalam game dari Hoyoverse khususnya Genshin Impact.

Pada halaman web ini terdapat beberapa bagian utama seperti navbar, main dan footer. Pada elemen nav terdapat judul web dan dua navigasi menuju home alias main dan about menuju bagian ini. Pada elemen main, terdapat beberapa bagian utama yaitu artikel dan user data alias aside dan about. Pada tampilan potrait mobile tepatnya ketika layar kurang dari 1200px, aside akan dipindah di bawah artikel agar lebar dari artikel dan aside tidak terlalu sempit. Selain itu nav dibuat fixed agar dapat diakses meskipun sedang berada di bagian bawah pada halaman. Pada footer terdapat beberapa icon yang menggunakan bantuan dari font-awesome.

Halaman web ini juga memanfaatkan Javascript dalam menyusun elemen dan style yang lebih dinamis. mulai dari konten pada artikel sehingga lebih efektif bahkan ukuran margin top yang digunakan pada elemen main agar tidak tertimpa oleh elemen header.
